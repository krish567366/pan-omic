# Installation Guide

## System Requirements

- Python 3.8+
- 8GB RAM minimum (16GB recommended)
- CUDA-capable GPU (optional, for acceleration)

## Quick Install

```bash
# Clone repository
git clone https://github.com/pan-omic/pce.git
cd pce

# Install in development mode
pip install -e .

# Verify installation
python -c "import pce; print('PCE installed successfully!')"
```
